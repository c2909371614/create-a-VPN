2019年最新ssr搭建梯子教程

技术标签： 搭建  梯子  ssr

前言：

        搭建srr服务器成功后，window/安卓/IOS/mac这些客户端都能通用，按照步骤来就一定能搭建成功。

1.首先做好准备工作(软件都放在百度云盘，自行下载)：

搭建思路：使用xshell连接你购买的服务器，执行代码==》搭建成功==》用客户端连接==》科学上网！

1.1需要的软件：链接：https://pan.baidu.com/s/104jhBqLFxy_GaI0nNjqyoQ 提取码：3po7

 如果使用的是苹果，可去ios商店搜索：Wingy、SSR等都可以，我用的是scanWingy
mac:  mac端代理工具。 

安卓客户端.7z：安卓客户端代理工具。 

SSTap：Windows客户端代理工具。 

Xshell：用来连接你的服务器的。
 

 1.2.一台外国的服务器(推荐vultr，目前性价比最高，各位可以去对比) 官方内部拿的链接，限时新活动，首次充值赠送 50$，支持支付宝微信支付等方式，如出现不能支付,请去掉对应的奖励即可

2.如果没购买服务器就参考这篇文章，包括所有的搭建方式

http://www.frogjun.com/fq-a/

官方服务器注册链接：https://www.vultr.com/?ref=7364687

 

 

3.SSR部署

  1. 下载xShell,上面有链接，vps信息在你刚刚购买的面板里面可以看到。打开xshell，新建链接，填写主机ip，端口号默认22端口不用填写，钩选重新链接，然后点连接。(要等服务器完全创建成功再连接，不然不会弹出输入用户名的框。如果连接不上，就是服务器的端口没有开启或者被墙了，在vultr先添加一个服务器，再删除当前服务器，直到能连上为止，vultr按小时计费，只要删除了就停止扣费)弹出“SSR安全警告”，选择“接受并保存”，用户名root（记得钩选记住用户名），密码在你的vultr里面去看，出现下图（root@vultr）代表成功，代表已经进入服务器了。

 

4.SSR搭建

安装SSR，复制执行以下命令(按照提示操作，输入自定义密码，如tj，然后回车，输入端口如8056然后回车，然后按任意键开始搭建 )
wget –no-check-certificate https://freed.ga/github/shadowsocksR.sh; bash shadowsocksR.sh




如提示wget :command not found请先执行



yum install wget -y



 



5.破解版锐速安装



一键更换内核脚本（Vultr需先执行此脚本）



wget -N –no-check-certificate https://freed.ga/kernel/ruisu.sh && bash ruisu.sh



脚本执行过程中，请勿进行任何操作。待服务器重启后，重新连接安装锐速即可。



锐速安装脚本



 



wget -N –no-check-certificate https://github.com/91yun/serverspeeder/raw/master/serverspeeder.sh && bash serverspeeder.sh



若提示：The name of network interface is not eth0, please retry after changing the name.请使用备用脚本备用脚本



wget -N –no-check-certificate https://raw.githubusercontent.com/91yun/serverspeeder/master/serverspeeder-all.sh && bash serverspeeder-all.sh



 



 



6.SStap配置



下载SStap,安装并填写SSR信息。按照下面的图填写即可，一共是六个参数，都是一一对应的，一个都不能少。如果出现请更换节点的提示，请先检查参数是否填写正确。参数确认填写正确还是连不上，没关系，那就删除服务器重新搭建吧！连接时注意选择跳过中国站点
